# SparklingSnakes

File metadata processor application. Used technology:
 - Python 3.11
 - PySpark 3.3.1
 - PostgreSQL 15.1

## Prerequisites

- [Ubuntu 22.04 LTS](https://releases.ubuntu.com/22.04/) (_Built on WSL_)
- [Docker](https://www.docker.com/)
- [Docker Compose](https://docs.docker.com/compose/)
